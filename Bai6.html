Cascading Style Sheets (CSS) ia a style sheet language used for describing the look and formating of a document written in a markup language. Althoungh most often used to change the style of web pages and user interfaces written in HTML and XHTML, the language can be applie to any kind of XML document, including plain XML, SVG and XUL.
